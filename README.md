

# 🗼 Tower of Hanoi

Tower of Hanoi game — playable in browser + Android APK. Built with pure HTML/CSS/JS, no frameworks.

🎮 **[Play Live →](https://hussainmotiwala.github.io/Tower-of-Hanoi/tower_of_hanoi.html)**
📱 **[Download Android APK →](https://github.com/HussainMotiwala/Tower-of-Hanoi/releases/tag/v1.0.0)**

---

## Features
- 3–10 rings with dynamic difficulty
- Drag & Drop + Click-to-Select control modes
- Move counter, timer, and personal best tracking
- Fully responsive — desktop, tablet, and mobile
- Landscape mobile layout support
- Confetti win animation 🎉

## How to Play
Move all rings from **Pole A** to **Pole C**.  
Rules: only one ring at a time, never place a larger ring on a smaller one.

## Tech Stack
- Vanilla HTML5 / CSS3 / (no frameworks)
- Google Fonts (Playfair Display, Inter)
- localStorage for best scores

## Run Locally
Just open `index.html` in any browser — no build step needed.
